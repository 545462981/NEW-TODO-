# 待办事项记事本

> iOS 简约风格 · 三列看板 · 拖拽操作 · 本地存储

一个基于 Electron 的 Windows 桌面待办事项管理工具，帮助你轻松追踪工作任务的进度。

---

## ✨ 功能

- **📋 三列看板** — 待办 / 进行中 / 已完成，一目了然
- **🖱️ 拖拽切换状态** — 把卡片拖到对应列即可改变任务状态
- **🔴🟡🟢 三级优先级** — 高/中/低，卡片左侧色条快速识别
- **📅 截止日期** — 过期任务自动标红提醒
- **📝 备注** — 每个任务可添加详细说明
- **💾 自动保存** — 每次操作自动写入本地 JSON 文件
- **⌨️ 键盘快捷键** — `ESC` 关闭弹窗，`Enter` 快速保存
- **🎨 iOS 简约风格** — 浅灰背景、白卡片、圆角阴影、流畅动画

## 🚀 快速开始

### 运行

下载 `dist/待办事项记事本.exe`，双击即可运行。无需安装，绿色便携。

### 开发

```bash
# 克隆仓库
git clone https://github.com/545462981/NEW-TODO-.git
cd NEW-TODO-

# 安装依赖
npm install

# 启动开发模式
npm start

# 构建 exe
npm run build
```

构建产物在 `dist/待办事项记事本.exe`。

## 💾 数据存储

所有任务数据保存在：

```
C:\Users\<用户名>\.todo-notepad\tasks.json
```

数据结构：

```json
[
  {
    "id": "mq4ji32owvrep8o",
    "title": "完成季度报告",
    "status": "todo",
    "priority": "high",
    "dueDate": "2026-06-15",
    "note": "需要包含 Q2 数据",
    "createdAt": "2026-06-08T01:36:12.048Z"
  }
]
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 唯一标识 |
| `title` | string | 任务标题 |
| `status` | enum | `todo` / `in-progress` / `done` |
| `priority` | enum | `high` / `medium` / `low` |
| `dueDate` | string | 截止日期（ISO 格式，可为空） |
| `note` | string | 备注（可为空） |
| `createdAt` | string | 创建时间（ISO 格式） |

## 🔧 自定义图标

1. 准备一张 256×256 或更大的 PNG 图片
2. 放到 `assets/icon.png`
3. 在 `package.json` 的 `build.win` 中加上 `"icon": "assets/icon.png"`
4. 运行 `npm run build` 重新打包
